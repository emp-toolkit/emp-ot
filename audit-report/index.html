emp-ot implementation audit

A high-level overview plus four implementation-level reports for an external auditor — one per protocol family. emp-ot is an oblivious-transfer library layered by a strict dependency order base OT → OT extension → silent OT / sVOLE: a handful of public-key bootstrap OTs (CO / PVW / CSW over P-256, BMM over ML-KEM-512), the IKNP / SoftSpoken symmetric extension over a shared streaming skeleton, the Half-Tree cGGM + MultiPointGadget + primal-LPN silent core driving Ferret / SilentFerret, and the subfield-VOLE family over GF(2128) and Fp. It builds on emp-tool::emp-tool and exposes a C++20 contract.
